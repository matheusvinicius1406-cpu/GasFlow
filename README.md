# GasFlow

Sistema operacional para depósitos de gás e água.

Objetivo:

- Automatizar atendimento
- Automatizar cadastro
- Automatizar pedidos
- Organizar entregas
- Integrar WhatsApp
- Gerenciar clientes
- Gerenciar estoque
- Gerenciar financeiro

Fluxo principal:

Cliente → Pedido → Entrega → Histórico
