# GasFlow - Visão do Projeto

## Missão

Automatizar completamente a operação de depósitos de gás e água.

O sistema deve reduzir tempo de atendimento, erros operacionais, sobrecarga da equipe e aumentar a velocidade das entregas.

---

## Público-Alvo

* Depósitos de gás
* Distribuidoras de água mineral
* Pequenos depósitos
* Médios depósitos

---

## Objetivos

* Cadastro inteligente de clientes
* Histórico completo de compras
* Integração com WhatsApp
* Aplicativo para entregadores
* CRM de clientes
* Controle operacional
* Controle financeiro
* Controle de estoque
* Automação de pedidos

---

## Identidade do Cliente

Todo cliente possui um código permanente.

Exemplo:

000001 = Rua da Paz Nº19 (Entre Arcanjo e Cajuí) (Maria)

O código nunca muda.

Mesmo que:

* Troque telefone
* Troque endereço
* Troque nome

---

## Fluxo Principal

Cliente → Pedido → Entrega → Histórico
