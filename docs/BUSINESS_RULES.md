# Regras de Negócio

## Cliente

Todo cliente possui código único.

Exemplo:

000001

O código nunca muda.

---

## Telefones

Um cliente pode possuir mais de um telefone.

---

## Pedidos

Todo pedido pertence a um cliente.

---

## Entrega

Todo pedido deve possuir status.

Status:

NOVO
PREPARANDO
EM_ENTREGA
ENTREGUE
CANCELADO

---

## Histórico

Nenhum pedido é apagado.

Nenhum cliente é apagado.

Apenas desativado.
